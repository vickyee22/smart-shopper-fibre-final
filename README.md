## Setup

1. Create a virtual environment:
    python3 -m venv venv
    source venv/bin/activate

2. Install dependencies:
    pip install -r requirements.txt

3. Add your OpenAI key:
    cp .env.example .env
    (edit .env to paste your API key)

4. Run:
    python chatbot.py

Then go to http://localhost:7860
